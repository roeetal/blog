Here is the list of ingredients. This recipe is not precise, and its not supposed to be.
Feel free to thrown in some chili or use whatever suitable vegetables or meats you have at home.

| ingredient                         |   quantity | unit   |
|------------------------------------|------------|--------|
| chicken thigh (boneless, skinless) |       4    |        |
| chicken seasoning                  |       4    | tsp    |
| carrot                             |       0.5  |        |
| zucchini                           |       0.5  |        |
| onion                              |       0.24 |        |
| cherry tomato                      |       4    |        |
| lemon                              |       0.5  |        |
| garlic                             |       1    | clove  |
| tomato puree                       |     750    | ml     |
| red wine                           |       1    | spritz |
| salt                               |            |        |
| sugar[^1]                          |       2    | tsp    |
| chicken soup broth                 |       1    | tsp    |
| cooking oil[^2]                    |            |        |
| basil                              |            |        |

Instructions:

1. Chop carrots and zucchini into small cubes, chop onion and cherry tomatoes finely.
2. Coat chicken thighs in seasoning and grill on medium heat. Set aside when done. [^3] 
3. In the same pan, fire up the veggies.
4. Cube lemon, mince garlic, chuck them into the pan with some stock.
5. Once the veggies are cooked, squeeze out lemon juice into pan. Add tomato puree, wine, sugar.
6. Blend (5) and put pack into pan.
7. Chop up chicken and add to the sauce.
7. Add salt and basil to taste.
